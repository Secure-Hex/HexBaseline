HexBaseline: checklist y evaluación de hardening para estaciones de trabajo.

Herramienta para validar el nivel de seguridad de endpoints (Windows, Linux y macOS): selecciona el sistema operativo, completa el checklist y genera un informe profesional listo para entregar.

Permite evaluar configuraciones críticas de seguridad como cifrado, privilegios, firewall, protección del endpoint, logging y reducción de superficie de ataque, alineado a buenas prácticas de ISO 27001, NIST y CIS.

Incluye ficha del activo auditado, seguimiento de cumplimiento por control y generación automática de informe en HTML optimizado para exportación a PDF.

Ideal para auditorías internas, evaluaciones de clientes, procesos de hardening y validación previa a pentesting.
